## a lightweighted LSH package on large document sets
### components
- LSH based on weighted Jaccard distance
- feature hashing
### dependencies
- pyfarmhash
- pymongo (if documents are indexed from a db)
- [libtopic-nlp-iit](https://github.com/vivi489/libtopic-nlp-iit)
